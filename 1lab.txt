rm: видаляє файл
    з ключем -r видаляє пусту директорію
    з ключами -rf видаляє заповнену директорію

mv: переміщяє, або перейменовує файлдерикторію
    перший аргумент назва директорії або директорії,
    другий куди перемістити і нову назву файладиректорії, якщо потрібно

cp: копіює файл
    з ключем -r може копіювати папки

ls: показує всі директорії і папки,
    з ключем -a показує приховані файли
    з ключем -l показує всі властивості файлів і 

la: аналогічно ls -a

ll: аналогічно ls -la

touch: створює файл

cat: виводить в консоль вміст файлу

nano: текстовий редактор
pwd - Working directory. Ця команда показує поточне находження в директорі
cd - Змінити директорію
touch - Створює файл
cat - Показує місткість файлу
mkdir - Створити папку echo "example"
mkdir dicetory - Створило директорію directory
example.txt - За допомогою цієї команди кинув текст "example" в текстовий файл 
example.txt rm - Видалити файл
